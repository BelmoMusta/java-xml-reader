  • Choix de l’application java :

  • Chois de l’application Angular :

– Angular est réputé et a une grande communité.

– La réutilisation des composants est un plus.

– La gestion des modules est garantie en toute flexibilité.

Dans ce sujet, on visera l’automatisation de la validation et l’extraction des informations de différents types de documents

Les types de documents concernés sont :

  • Les pièces d’identité : CIN, Passeport, Carte de séjour, Permis de conduite …

  • Les justificatifs de domicile : tout document contentant l’adresse de résidence d’une personne (une entreprise ), comme, les relevé e paiement des facture d’électricité, eau potable, téléphone/internet …

  • Les relevés d’identité bancaires

Le livrable de ce sujet sera une application java web qui permet l’interaction homme-machine, tout en fournissant, des services REST équivalents.

Validation

Le processus de validation sert à détecter automatiquement si les informations saisies de la personne en question sont équivalentes à ce qui est lu sur les documents, Exemple :
